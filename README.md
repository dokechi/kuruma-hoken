# 自動車保険タイプ診断

保険証券なしで20問に答えるだけで、ユーザーの「事故時の不安タイプ」と、5社の中で最初に比較すべき保険会社タイプを整理する静的サイトです。

## サイト概要

このサイトは、保険料や補償額を比較するものではありません。車の使い方、運転する人、事故時に不安なことから、自動車保険を見るときの軸を整理します。

## 目的

- 保険証券なしで、自分の保険選びの軸を確認する
- 証券・等級・保険料・更新月などを入力せずに診断する
- 個人情報を入力せず、ブラウザ内だけで完結する
- 特定保険商品の加入をすすめず、比較の観点を整理する

## 診断でできること

- 20問の選択式質問に回答する
- 6タイプすべての「タイプ一致度」を確認する
- 事故時の不安がどのタイプに近いかを見る
- 5社の中では、まずどの型を比較する理由があるかを確認する
- 価格重視の場合に、ダイレクト型や一括見積りも選択肢として確認する

## 診断でできないこと

- 特定保険商品の加入推奨
- 保険料の見積り
- 補償額や特約の個別比較
- 契約可否の判断
- 実際の契約内容の確認

## ファイル構成

```text
.
├── index.html
├── style.css
├── script.js
├── data
│   ├── questions.json
│   ├── scoring.json
│   └── results.json
└── README.md
```

- `index.html`: トップ、質問、結果サマリー、詳細理由、次のアクション、出典・参考資料を持つ単一ページ
- `style.css`: 白背景とピンクを基調にしたレスポンシブデザイン
- `script.js`: 画面切り替え、回答管理、スコア計算、タイプ一致度表示
- `data/questions.json`: 20問の質問データ
- `data/scoring.json`: 全20問分の回答別スコア
- `data/results.json`: 6タイプの表示文言と理由

## GitHub Pagesでの公開方法

1. このリポジトリをGitHubへpushします。
2. GitHubのリポジトリ画面で **Settings** を開きます。
3. **Pages** を選択します。
4. **Build and deployment** のSourceで **Deploy from a branch** を選びます。
5. Branchで公開したいブランチと `/ (root)` を選択して保存します。
6. 数分後に表示されるURLへアクセスします。

静的ファイルのみで構成しているため、ビルド手順は不要です。

## 免責事項

この診断は、特定の保険商品の加入をすすめるものではありません。結果は、回答内容から見た「事故時の不安タイプ」と各社の特徴の近さを示すものです。実際の契約判断には、補償内容・保険料・車両条件などの確認が必要です。
