# 自動車保険モデルケース比較

保険証券や個人情報を入力せずに、モデルケースから自分に近い車の使い方を選び、自動車保険を見るときのポイントを整理する静的サイトです。

## サイト概要

このサイトは、保険料や補償額を比較するものではありません。車種・年代・使い方ごとのモデルケースから、自分に近い車の使い方を選び、事故時に不安になりやすい点や、保険を見るときの軸を確認するためのサイトです。

トップページでは「モデルケースから確認」を主導線にしています。近いケースを選ぶと、見落としやすいポイント、まず比較する理由があるタイプ、他タイプだとズレやすい点を確認できます。

## 目的

- モデルケースから、自分に近い車の使い方を選ぶ
- 保険証券なしで、自分の保険選びの軸を確認する
- 証券・等級・保険料・更新月などを入力せずに確認する
- 個人情報を入力せず、ブラウザ内だけで完結する
- 入力内容や回答データを保存しない
- 特定保険商品の加入をすすめず、比較の観点を整理する

## モデルケースでできること

- 車種・年代・使い方が近いケースを選ぶ
- 事故時の不安や見落としやすいポイントを確認する
- まず比較する理由があるタイプを確認する
- 他タイプだと見るべき軸がズレやすい点を確認する
- 保険会社の優劣ではなく、車の使い方との相性を整理する

## 詳しく診断したい人向けの20問診断

20問診断は、モデルケースだけでは判断しにくい人や、より詳しく自分の不安タイプを整理したい人向けの補助機能です。

20問診断では、次の内容を確認できます。

- 20問の選択式質問に回答する
- 6タイプすべての「タイプ一致度」を確認する
- 事故時の不安がどのタイプに近いかを見る
- 5社の中では、まずどの型を比較する理由があるかを確認する
- 価格重視の場合に、ダイレクト型や一括見積りも選択肢として確認する

## このサイトでできないこと

- 特定保険商品の加入推奨
- 保険料の見積り
- 補償額や特約の個別比較
- 契約可否の判断
- 実際の契約内容の確認

## データの扱い

- 保険証券は不要です
- 個人情報の入力は不要です
- 回答内容や選択内容は保存しません
- サーバーへデータを送信せず、ブラウザ内で完結します

## ファイル構成

```text
.
├── index.html
├── style.css
├── script.js
├── data
│   ├── questions.json
│   ├── scoring.json
│   └── results.json
└── README.md
```

- `index.html`: トップ、モデルケース一覧、モデルケース詳細、20問診断、結果サマリー、詳細理由、次のアクション、出典・参考資料を持つ単一ページ
- `style.css`: 白背景とピンクを基調にしたレスポンシブデザイン
- `script.js`: 画面切り替え、モデルケース表示、回答管理、スコア計算、タイプ一致度表示
- `data/questions.json`: 詳しく診断したい人向けの20問の質問データ
- `data/scoring.json`: 全20問分の回答別スコア
- `data/results.json`: 6タイプの表示文言と理由

## GitHub Pagesでの公開方法

1. このリポジトリをGitHubへpushします。
2. GitHubのリポジトリ画面で **Settings** を開きます。
3. **Pages** を選択します。
4. **Build and deployment** のSourceで **Deploy from a branch** を選びます。
5. Branchで公開したいブランチと `/ (root)` を選択して保存します。
6. 数分後に表示されるURLへアクセスします。

静的ファイルのみで構成しているため、ビルド手順は不要です。

## 免責事項

このサイトは、特定の保険商品の加入をすすめるものではありません。モデルケースや診断結果は、車の使い方や事故時の不安から、保険を見るポイントを整理するものです。実際の契約判断には、補償内容・保険料・車両条件などの確認が必要です。
